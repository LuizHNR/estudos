Muitos artigos longos têm subtítulos para ajudá-lo a acompanhar a estrutura do que está sendo escrito. Pode até haver sub-subtítulo (ou título de nível inferior)

Many long articles have sub-headings so to help you follow the structure of what is being written. There may even be sub-sub-heading (or lower-level heading).

Tente não utilizar a tag br, pois ela foge da norma

  • O Atributo scr: indica indica o caminho da imagem, local onde o arquivo está armazenado. Você deverá declarar a pasta e o nome correto da imagem desejada com a sua respectiva extensão.
  • Atributo alt: insere um pequeno texto que descreve a imagem, ele será exibido caso a imagem não seja carregada pelo navegador. Esse atributo também ajudará na acessibilidade a pessoas que tenham alguma deficiência visual e façam uso de leitores de tela para navegar pela Internet. Quando o leitor encontrar a imagem irá procurar o conteúdo desse atributo e fazer sua leitura. O uso do atributo alt, é uma boa prática.
  • Atributo title: é um texto que será exibido assim que você posicionar o ponteiro do mouse sobre a imagem. Ele não é exclusivo para as imagens, praticamente todos os elementos HTML podem usar. Na maioria das vezes o valor atribuído a esse atributo é o mesmo utilizado pelo atributo alt.
